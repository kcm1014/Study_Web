left : 기준 위치와 요소 사이에 왼쪽이 얼마나 떨어져 있는지를 지정
right : 기준 위치와 요소 사이에 오른쪽이 얼마나 떨어져 있는지를 지정
top : 기준 위치와 요소 사이에 위쪽이 얼마나 떨어져 있는지를 지정
bottom : 기준 위치와 요소 사이에 아래쪽이 얼마나 떨어져 있는지를 지정

position : 웹 요소의 위치를 지정
    static : 문서의 흐름에 맞춰 배치한다. 기본값
    relative : 위치값을 지정할 수 있다는 점을 제외하면 static과 같다.
    absolute : relative 값을 사용한 상위 요소를 기준으로 위치를 지정 배치한다.
    fixed : 브라우저 창을 기준으로 위치를 지정 배치한다.
